# Calculadora Web

Calculadora cientifica desarrollada con HTML, CSS y JavaScript puro. La aplicacion ofrece operaciones basicas y funciones matematicas comunes en una interfaz limpia, responsive y lista para abrirse directamente en el navegador.

## Vista general

Este proyecto implementa una calculadora web con soporte para:

- Operaciones basicas: suma, resta, multiplicacion, division y porcentaje.
- Funciones cientificas: `sin`, `cos`, `tan`, `log`, `ln`, `sqrt`, potencias y factorial.
- Constantes matematicas: `pi` y `e`.
- Cambio de modo angular entre `DEG` y `RAD`.
- Entrada por botones y tambien por teclado.
- Interfaz adaptable para escritorio y celular.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Como usarlo

1. Abre [calculadora/index.html](calculadora/index.html) en tu navegador.
2. Ingresa operaciones con los botones o con el teclado.
3. Presiona `=` o `Enter` para calcular.

## Atajos de teclado

- `0-9`: numeros
- `+ - * / % ^`: operadores
- `.` o `,`: decimal
- `Enter` o `=`: calcular
- `Backspace`: borrar ultimo caracter
- `Escape`: limpiar

## Funciones disponibles

- `sin(x)`
- `cos(x)`
- `tan(x)`
- `log(x)`
- `ln(x)`
- `sqrt(x)`
- `x^y`
- `x!`
